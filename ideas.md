# UNITS|SG Portal — Design Brainstorm

We are porting the existing /sg portal from the atomium1-goldensource codebase. The design system is already established (dark navy theme, SG color palette, glassmorphism cards). Our task is to enhance it with motionsites.ai-inspired motion design. Here are three approaches:

<text>
## Idea 1: "Sovereign Pulse" — Living Infrastructure

**Design Movement:** Bioluminescent Futurism — inspired by deep-sea organisms and neural networks
**Core Principles:** Organic glow, breathing rhythms, living data, ambient intelligence
**Color Philosophy:** The existing SG palette (navy #0A1628, teal #00A3A1, orange #EF7C00, red #EE2536) treated as bioluminescent light sources against deep ocean darkness. Colors pulse and breathe rather than sitting static.
**Layout Paradigm:** The existing single-column with sidebar is preserved, but every section breathes. Content fades in with staggered timing. Cards have living borders that pulse on hover.
**Signature Elements:** (1) Glowing orb behind the orchid logo that pulses like a heartbeat, (2) Liquid glass card borders that shimmer on hover with gradient sweeps, (3) Floating particle connections between sections suggesting a network
**Interaction Philosophy:** Every hover reveals life — cards glow, borders shimmer, icons pulse. Scrolling triggers organic reveals, not mechanical slides.
**Animation:** Slow breathing pulses (4-8s cycles), staggered fade-ins (100ms delays between siblings), smooth counter animations on scroll, parallax depth on hero backgrounds
**Typography System:** Inter for body (already in use), but with careful weight hierarchy — 300 for large display text, 400 for body, 600 for emphasis, 700 for section numbers

## Idea 2: "Digital Sovereignty" — Precision Engineering

**Design Movement:** Swiss Brutalist Digital — clean grids, sharp edges, mathematical precision
**Core Principles:** Precision, clarity, authority, institutional confidence
**Color Philosophy:** Same palette but used with surgical precision. Teal for primary actions, orange for warnings/highlights, red for critical markers. No gradients — flat, confident color blocks.
**Layout Paradigm:** Strict 12-column grid. Content snaps into place with mechanical precision. No organic curves — everything is angular and deliberate.
**Signature Elements:** (1) Horizontal rule animations that draw across the screen as section dividers, (2) Counter numbers that flip like departure boards, (3) Cards with sharp 1px borders that highlight on hover with color fills
**Interaction Philosophy:** Clicks and hovers produce immediate, crisp responses. No easing — linear transitions. The interface feels like a precision instrument.
**Animation:** Linear transitions (200ms), number flip counters, horizontal line draws, snap-in reveals with no bounce
**Typography System:** Mono-spaced for data (JetBrains Mono), Inter 500 for body, Inter 700 for headers. All caps for labels.

## Idea 3: "Cinematic Infrastructure" — Premium Motion Design

**Design Movement:** Motionsites.ai-inspired Cinematic Web — premium SaaS meets financial infrastructure
**Core Principles:** Cinematic depth, premium motion, ambient atmosphere, narrative scroll
**Color Philosophy:** The SG palette used as light sources in a dark environment. Teal and cyan create ambient atmosphere. Orange and amber provide warmth and urgency. Red marks critical infrastructure. Purple adds depth and sophistication.
**Layout Paradigm:** Preserve the existing sidebar + content layout, but add cinematic depth layers — mesh gradient backgrounds that slowly drift, subtle particle fields, and layered z-depth on cards.
**Signature Elements:** (1) Triple-layer glowing orb behind the orchid logo with expanding sonar rings, (2) Liquid glass cards with gradient border sweeps and backdrop blur, (3) Animated mesh gradient backgrounds on deep-dive pages that slowly morph between palette colors
**Interaction Philosophy:** The page tells a story as you scroll. Each section reveals with purpose. Hover states provide tactile feedback — cards lift, borders glow, content shifts. The experience feels like navigating a premium financial terminal.
**Animation:** Breathing glows (6s ease-in-out), staggered reveals (150ms delays), smooth cubic-bezier counters (2s duration), mesh gradient morphing (20s infinite), card hover lifts (300ms spring), border gradient rotation (3s linear infinite)
**Typography System:** Inter 300 for hero display (large, light, elegant), Inter 400 for body, Inter 600 for section headers, Inter 700 for data callouts. Tracking widened on labels (0.2em).

## Selected Approach: Idea 3 — "Cinematic Infrastructure"

This is the natural choice because:
1. It directly incorporates the motionsites.ai elements we prototyped and Rajeev approved
2. It preserves the existing design system while elevating it
3. It matches the premium, institutional tone required for a financial infrastructure blueprint
4. The 5 specific enhancements (glowing orb, liquid glass, counters, scroll reveals, cinematic backgrounds) map directly to this philosophy
